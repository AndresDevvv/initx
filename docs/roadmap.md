# Roadmap

Status: alpha.

This file exists to separate shipped functionality from ideas.

Implemented today:

- `initx python <name>`
- `initx python .`
- `initx node <name>`
- `initx node .`

Not implemented yet:

- `initx rust <name>`
- `initx go <name>`
- more template options and richer starter files
- platform-specific packaging and install validation

Anything in the "Not implemented yet" list is a TODO, not a supported command.
